# Sample petitions

## `synthetic-eb1a-petition.pdf`

A **synthetic** EB-1A petition generated by `scripts/generate-sample-pdf.mjs`. The
petitioner is fictional. Every page is watermarked SYNTHETIC.

Designed with deliberate weaknesses so the RFE Sentinel finds meaningful issues:

| Weakness baked into the petition | Should surface as |
|---|---|
| Petitioner cites a Student Council membership where the parent association doesn't itself demand outstanding achievement | LLM flags the membership criterion |
| Claims judging via two manuscript reviews and one program committee — all >3 years ago | LLM flags as marginal |
| Original-contributions claim relies in part on a recommender who was the Ph.D. advisor | LLM flags independence concern |
| Salary `$215,000` cited with no BLS / prevailing-wage comparator | Heuristic + LLM flag |
| Only ~14 citations across 2 publications | Heuristic flag (`low-citation-count`) |
| Only 3 recommendation letters | Heuristic flag (`few-recommenders`) |

Regenerate after edits to the script:

```bash
node scripts/generate-sample-pdf.mjs
```
